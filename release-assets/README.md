# QuantumLang for VS Code Release Assets

Official QuantumLang extension assets by Quantum Technology.

Optional release-note assets live here.

## Screenshot Approval

Screenshots are optional for Q80 and must be approved before they are committed
or referenced from marketplace-facing release notes. Until approval happens,
keep this directory as guidance only and do not add binary screenshots.

Recommended screenshots for a packaged release:

- `preflight.png`: `QuantumLang: Preflight` output showing qtlc path, version,
  package root, target, and cache health.
- `diagnostics.png`: diagnostics from `qtlc server` in an installed extension
  workspace.
- `cache-dashboard.png`: `QuantumLang: Cache Dashboard` quick-pick.
- `qn-file-icons.png`: Explorer view showing the QuantumLang `.qn` dark/light
  file icon theme.

Keep screenshots optional and small. Do not put compiler logic, generated
artifacts, or local machine paths in release assets.
